# Welcome to Orches

Orches is an open-source platform for building and running teams of AI agents. Agents collaborate, delegate tasks, use tools, and write results here — in this workspace.

---

## Getting started

### 1. Talk to the main agent

Open the **Orches** chat tab and type a message. Start simple:

- *"What can you do?"*
- *"Research the latest news on AI agents"*
- *"Create a team of agents to write a blog post"*

The main agent will answer directly or delegate to a specialist.

### 2. Create your first agent

Click **+ New Agent** in the top bar. Give it a name, a system prompt, and a set of tools. Then add it to the main agent's **CAN DELEGATE TO** list so it can receive tasks.

Or just ask the main agent:
> *"Create a researcher agent that searches the web and writes summaries to the workspace"*

### 3. Use the workspace

This folder is the shared workspace — agents read and write files here, and so can you.

- Upload files by dragging them into the chat
- Ask an agent to read a file: *"Analyze report.csv"*
- Open any file from the **Workspace** tab in Canvas

Files agents create appear here automatically.

### 4. Connect external tools (MCP)

Go to **Settings → MCP Servers** to connect GitHub, Notion, Linear, and more. Each connected server adds new tools that any agent can use.

---

## Tips

- **EventLog** (left panel) shows every tool call and step in real time
- **Canvas** shows rich output — markdown reports, code, browser, files
- Agents can create other agents — ask the main agent to build a full team
- Scheduled tasks run automatically — set them in agent settings under **Tasks**

Happy building!
